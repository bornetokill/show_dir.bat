```
SYNTAX
"show_dir.bat"   "[path]" options

    options
        v show visible only
        h show hidden only
        d show directories
        f show files
    options must be combined like Eg. dv
    in any order
Example 1 :- "show_dir.bat" C:\ dh
Example 2 :- "show_dir.bat" "C:\folder 1" fdv
```
